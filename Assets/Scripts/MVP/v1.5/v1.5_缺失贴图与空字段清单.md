## 五、场景组件空引用（非 Sprite，但影响表现）

| 组件 | 字段 | 状态 | 影响 |
|------|------|------|------|
| `DurianOpener` | `shareButtonGroup` | ❌ null | 爆肉/连击≥3 时「炫耀一下」按钮不显示（逻辑已有） |
| `KnifeTool` | `audioSource` | ❌ null | 划刀开壳无音效 |
| `KnifeTool` | `crackClip` | ❌ null | 划刀开壳无裂壳音效 |
| `SellPage` | `coinParticleRoot` | ❌ null | 卖出时金币粒子用代码动态创建根节点（可工作，未预制） |
| `ShopPage` | `upgradeEffectRoot` | ❌ null | 商店升级光柱特效未挂节点 |
| `MarketPage` | `refreshButtonText` | ❌ null | 「换一批」按钮可能无文字（图标仍可用） |

**分享按钮占位：** 在 `OpenPage` 下建 `ShareButtonGroup` + Button「炫耀一下」，拖到 `DurianOpener.shareButtonGroup`。

**裂壳音效：** 在 `KnifeTool` 上挂 `AudioSource`，并指定 `AudioClip`（项目内暂无标准裂壳资源，需自备或从 MVP 音效目录添加）。
