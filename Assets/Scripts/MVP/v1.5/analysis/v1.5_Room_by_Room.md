## 四、Unity 实现

每个房位是一个独立的 RoomSlot Prefab：

```
RoomSlot_P1 (RectTransform, 位置=(0, -80))
  ├── CoverImage (RC-JZ-P1, 128×128) ← 初始显示，捅开后弹飞
  └── FleshImage (RF-XX, 128×128)    ← 初始隐藏，捅开后显示

RoomSlot_P2 (RectTransform, 位置=(-110, -30))
  ├── CoverImage (RC-JZ-P2)
  └── FleshImage (RF-XX)
...以此类推 P3/P4/P5
```

**5个壳盖各自独立贴图，5个果肉引用同一组 RF-00~100**。

壳盖弹飞动画：
```csharp
coverImage.rectTransform.DOAnchorPos(coverImage.rectTransform.anchoredPosition + Random.insideUnitCircle * 80f, 0.3f)
    .SetEase(Ease.OutQuad);
coverImage.DOFade(0f, 0.2f).OnComplete(() => coverImage.gameObject.SetActive(false));
fleshImage.gameObject.SetActive(true);
```

## 五、逐房揭示流程

```
1. D-01 完整榴莲显示（全局背景）
2. 5个 RoomSlot 叠在对应位置上，Cover 显示，Flesh 隐藏
3. 玩家在某房盖上滑动 → Cover 弹飞 → Flesh 显示对应RF级别
4. 每开一房：
   - 估价更新 = f(已开房满肉比例 × 剩余房数 × 不确定性)
   - 第一个房开后 [见好就收] 按钮出现
5. 中途卖 → 按当前估价结算
6. 5房全开 → 评级弹出
```
