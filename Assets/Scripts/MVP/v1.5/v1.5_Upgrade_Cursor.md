# 《榴莲开了》v1.2 → v1.5 统一升级指令

> **文档版本**：v1.5.1 | 2026-06-26  
> **基线代码**：v1.2（市场 / 背包 / 售卖 / 商店 Lv.2 / CP 开裂动画）  
> **目标版本**：v1.5（逐房揭示 + 中途卖 + 图鉴 + 连击视觉 + 每日目标 + 商店 Lv.3 + 分享）  
> **来源**：融合 `v1.5.0_Upgrade_Cursor(1).md` 与 `v1.5_Cursor_Prompts.md`，修正步骤依赖顺序  

**使用方式**：按下方「执行顺序」逐步执行；每步将 **Cursor 提示词** 块贴入 Cursor。贴图对照见 `v1.5_Texture_Prompts.md`，资源目录 `Assets/Art/v1.5texture/`。

## 一、v1.5 相对 v1.2 改了什么

| 模块 | v1.2 | v1.5 |
|------|------|------|
| 外观 | 9 张外观变体贴图（D-A） | **1 张基底 + `Image.color` 调色** |
| 开果 | 滑动 + CP-01~05 + 半壳划开 + D-O 已开图 | **5 房 RC 壳盖 + RF 果肉，逐房捅开** |
| 开果交互 | 一刀划完全部 | 每房可点/划开；首房后 **见好就收** 中途卖 |
| 进度 | 无 | **图鉴 15 格**（3 品种 × 5 档位） |
| 反馈 | 无 | **连击火焰**（纯视觉，不改售价） |
| 日常 | 无 | **每日目标** 赚 500 金 → 奖励 50 金 |
| 商店 | Lv.2（+20%） | **Lv.3**（500 / 1500 金，+20% / +35%） |
| 社交 | 无 | 爆肉或连击≥3 → **炫耀分享**（微信 MVP 占位） |

**首期品种范围**：逐房揭示 **仅金枕 5 房**（RC-JZ-P1~P5）。干尧/猫山王可用 D-02/D-03 做市场调色，壳盖 RC 后续追加。

## Step 4：StreakCounter（连击视觉）

### Cursor 提示词

```
创建 Assets/_Project/Scripts/System/StreakCounter.cs：

public class StreakCounter
{
    public int CurrentStreak { get; private set; } = 0;

    public void OnRoomRevealed(bool hasFlesh)
    {
        if (hasFlesh) CurrentStreak++;
        else CurrentStreak = 0;

        if (CurrentStreak > PlayerProgression.Instance.BestStreak)
            PlayerProgression.Instance.BestStreak = CurrentStreak;

        EventBus.Publish(new StreakUpdatedEvent { Combo = CurrentStreak });
    }

    public int GetFlameLevel() => CurrentStreak switch {
        < 3 => 0, < 5 => 1, < 7 => 2, < 10 => 3, _ => 4 };

    public void Reset()
    {
        CurrentStreak = 0;
        EventBus.Publish(new StreakUpdatedEvent { Combo = 0 });
    }
}

在 GameLifetimeScope.Configure 注册：builder.Register<StreakCounter>(Lifetime.Singleton);
```

## Step 7：重写 DurianOpener + OpenPage

### 目标
废弃 v1.2 的 CP 叠加、半壳划开、底部房位弹出；改为 `PrepareOpen` + 逐房交互 + 中途卖。

### Cursor 提示词

```
完全重写 Assets/_Project/Scripts/Component/DurianOpener.cs（参考 v1.5.0 Step 4，并注入以下依赖）：

[Inject] DurianSpriteConfig, ShopManager, StreakCounter
[SerializeField] Image wholeDurianImage;           // D-01 + 调色
[SerializeField] Transform roomSlotsParent;        // 768×768 居中容器
[SerializeField] GameObject roomSlotPrefab;
[SerializeField] DurianRoomConfig roomConfig;      // 或内置 roomPositions[]
[SerializeField] GameObject sellMidwayGroup;      // 见好就收栏
[SerializeField] Text sellMidwayPriceText;
[SerializeField] Button sellMidwayButton, continueButton;
[SerializeField] Image ratingBadgeImage;
[SerializeField] CanvasGroup ratingCanvasGroup;
[SerializeField] GameObject shareButtonGroup;    // 炫耀一下

核心流程：
- PrepareOpen(durian)：显示 wholeDurianImage（sprite+color）；AssignRoomGrades(yieldRate) 并打乱；实例化 5×RoomSlot(RC[i], RF[grade[i]])
- OnRoomOpened：连击、UpdateEstimate、首房后显示 sellMidwayGroup、5 房满 → OnAllRoomsOpened
- OnSellMidway：发布 DurianMidwaySoldEvent，快速开剩余房，OpenPage 跳 SellPage 按估价结算
- OnAllRoomsOpened：UnlockCollection、ShowRating(GetRatingIcon)、条件显示分享
- Reset()：清理 RoomSlot、隐藏 UI

AssignRoomGrades：fullRooms = Round(yieldRate*5)，前 fullRooms 个 Perfect 其余 Empty，再 Fisher-Yates 打乱。

UpdateEstimate：根据已开房满肉比例 + 剩余房不确定性估算金币，显示在 sellMidwayPriceText（绿涨红跌 + PunchScale）。

修改 OpenPage.cs：
- Show(durian) 调用 durianOpener.PrepareOpen(durian) 替代旧 Reset+KnifeTool 为主流程
- 订阅 StreakUpdatedEvent → 更新 comboDisplay（combo≥2 显示 CO-01 + 数字）
- 绑定 sellMidwayButton / continueButton
- 弱化或移除 KnifeTool 主流程（可保留「继续开」一键开剩余房，或废弃滑动开果）
- 中途卖：监听 DurianMidwaySoldEvent → ShowSell(估价)

OpenPage Canvas 需有（编辑模式 Tools/llopen 或手动）：
- wholeDurianImage（与 DurianImage 可同节点）
- roomSlotsParent
- sellMidwayGroup（估价 + 见好就收 + 继续开）
- ratingBadgeImage + ratingCanvasGroup
- comboDisplay（连击火焰 CO-01 + 数字）
```
