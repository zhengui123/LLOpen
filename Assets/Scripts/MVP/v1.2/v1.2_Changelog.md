## 二、OpenPage 开果动画 —— 重做

### 2.1 当前问题

**现状（v1.0 错误实现）**：
```
┌─────────────────────────┐
│                         │
│     [榴莲完整图]          │  ← 榴莲在中间
│                         │
│  滑动后 ↓               │
│                         │
│  [左半壳]    [右半壳]     │  ← 壳飞到下方，方向错误
│  ←飞走         飞走→     │
│                         │
└─────────────────────────┘
```

**问题分析**：
1. 左右壳从果的位置向下弹到画面底部，视觉上壳和果分离了，不自然
2. 裂壳应该是**自上而下**的刀切效果，不是左右飞开
3. 壳裂开的过程应该是**逐级渐进**的，不是一刀瞬间完成

### 2.2 修正设计

**正确方案**：
```
┌─────────────────────────┐
│                         │
│     [手指滑动手势]        │  ← 引导在榴莲顶部
│  ┌───────────────────┐  │
│  │                   │  │
│  │   [榴莲完整图]      │  │  ← 果始终在中间
│  │         ↑          │  │
│  │    [开裂叠加层]     │  │  ← 壳开裂效果叠加在果上方
│  │                   │  │
│  └───────────────────┘  │
│                         │
│  滑动后 → 开裂逐级变化   │
│                         │
└─────────────────────────┘
```

**动画流程（修正后）**：

| 阶段 | 贴图 | 画面状态 | 触发条件 |
|------|------|---------|---------|
| 阶段0 | D-01~03 完整榴莲 | 榴莲居中，壳完整，顶部有滑动手势引导 | 进入OpenPage |
| 阶段1 | CP-01 裂纹初现 | 榴莲居中，顶部出现细裂纹线，壳微微张开 | 滑动20% |
| 阶段2 | CP-02 裂纹扩大 | 裂纹加深加宽，壳开始向两侧微微分离 | 滑动40% |
| 阶段3 | CP-03 半开裂 | 壳明显分离，中间露出部分金黄色果肉 | 滑动60% |
| 阶段4 | CP-04 大半开裂 | 壳大部分剥落，果肉大面积露出 | 滑动80% |
| 阶段5 | CP-05 + D-O 已开榴莲 | 壳完全剥落消失，转为已开榴莲图，果肉尽显 | 滑动100% |
| 结算 | R-01~06 评级图标 | 评级弹窗，出肉率数字跳动 | 动画完成后0.5秒 |

**关键改动**：
- 废弃 SH-01（左半壳）和 SH-02（右半壳）
- 新增 CP-01~CP-05（开裂阶段叠加图）
- 开裂图始终在榴莲完整图的**上方**叠加显示
- 过渡用 DOTween 的 CrossFade 或序列帧切换
- 榴莲**始终在画面中央**，不移动

## 三、其他页面优化分析

### 3.1 MarketPage 市场页

| 问题 | 当前状态 | v1.2 优化 |
|------|---------|----------|
| 3颗榴莲无法重roll | 选完品种后3颗固定，不满意只能退出重进 | 增加"换一批"按钮（消耗50金币/看广告） |
| 品种切换无确认 | 点品种直接刷新，误触无法撤销 | 品种按钮点击后高亮，需再次确认才刷新 |
| 金币信息展示较弱 | 顶部小金币数字，不够醒目 | 金币数字加大，颜色金色，增加闪烁动画 |
| 不满意的榴莲无反馈 | 点了不喜欢只能无视 | 增加"不满意"按钮，可标记某颗不看好的榴莲 |

### 3.2 SellPage 售卖页

| 问题 | 当前状态 | v1.2 优化 |
|------|---------|----------|
| 无售卖反馈动画 | 点卖出→金币直接加，无动画 | 增加金币飞出动画（从榴莲飞向顶部金币栏） |
| 出肉率回顾过于简单 | 只显示一个数字 | 显示每房结果回顾小图 + 评级图标 |
| "看广告加价"按钮不明显 | 普通按钮样式 | 按钮加闪烁边框特效，吸引点击 |

### 3.3 BagPage 背包页

| 问题 | 当前状态 | v1.2 优化 |
|------|---------|----------|
| 空背包无引导 | 空白页面，不知道该干嘛 | 增加"去市场买榴莲吧"插图 + 快捷跳转按钮 |
| 榴莲卡片信息太简 | 只有品种名+外观图标 | 增加出肉率预览（开之前是"？"，吸引好奇心） |

### 3.4 ShopPage 商店页

| 问题 | 当前状态 | v1.2 优化 |
|------|---------|----------|
| 去广告卡入口不明显 | 底部小文字 | 改为独立卡片样式，增加"首充优惠"角标 |
| 升级后无庆祝动画 | 直接数字变化 | 增加升级特效（光柱+音效） |

### 3.5 全局改进

| 改进点 | v1.0 状态 | v1.2 优化 |
|--------|----------|----------|
| 页面过渡 | 直接切换，生硬 | 增加 Slide In/Out 过渡动画（0.3秒） |
| 返回按钮一致性 | 位置/样式不统一 | 统一左上角位置，统一大小（64×64） |
| 错误处理 | 无 | 贴图缺失时显示默认占位图 + 控制台警告 |
| 音效反馈 | 未接入 | 新增划刀音效（切壳声）、出肉音效（惊喜声）、金币音效 |
