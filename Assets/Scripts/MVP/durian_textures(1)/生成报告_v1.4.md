## 四、已知问题

### 4.1 AI 水印

部分图片底部有"图片由AI生成"水印。如需去除，建议：
- 用 Photoshop 或同类工具裁切/克隆修复底部边缘
- 或在 Unity 中按 Sprite 裁剪时底部留出裁切裕量

### 4.2 透明背景

ImageGen 的 `background: transparent` 参数在执行中未完全生效。建议：
- 在 Photoshop 中用魔棒/色彩范围选中白色背景后删除
- 或在 Unity 中将 Sprite 的 Texture Type 设为 Sprite(2D and UI)，调整 Alpha 通道

### 4.3 品种内一致性

本次使用纯文本提示词 + 身份指纹法生成。由于 ImageGen 不支持图生图模式，品种内各变体的像素级一致性无法保证。

**如果一致性不达标**（对照文档第六章验证标准），建议：
- 选取品种基准图（D-01v4/D-02v4/D-03v4）效果最佳的作为主参考
- 使用时用手动 PS 调色 / 添加裂纹 / 粒子特效来模拟变体
- 或换用支持 img2img 的模型重新生成变体

### 4.4 开裂序列连续性

CP-01→05 每帧独立生成，序列间的壳型、刺型、斑点位置无法做到像素对齐。如需精确逐帧动画，建议在生成满意 CP-01 后，用 PS 逐帧手动扩大裂缝。

## 五、后续建议

1. **筛选**：每种品类选出视觉效果最好的一张作为"真实版本"，其余可放弃
2. **去水印 + 去背景**：用批量 PS 动作处理所有 25 张
3. **导入 Unity**：Texture Type → Sprite(2D and UI)，Mesh Type → Tight，调整 Pixels Per Unit
4. **游戏内测试**：在 GTest 项目中实际验证各贴图的视觉效果
