# 《榴莲开了》v1.3 功能改动文档 —— 目标系统与爽点层

> 版本：v1.3 | 日期：2026-06-23
> 基线：v1.2（已完成开果动画修正 + 页面优化，但缺少长期游玩目标）
> 核心问题：玩家买→开→卖→循环，但没有"为了什么"，缺乏留存钩子

---

## 一、问题诊断

| 问题 | 表现 | 根因 |
|------|------|------|
| 无升级目标 | 商店升到Lv.2就没追求了 | 缺乏多级进度系统 |
| 无金币出口 | 金币攒了不知道花在哪 | 缺乏消耗金币的长期系统 |
| 无收集驱动 | 开完就卖了，不记得开过什么 | 缺乏图鉴/记录系统 |
| 无惊喜爆发 | 出肉率好就是数字跳一下 | 缺乏连击/暴击等随机爽点 |
| 无每日理由 | 没有"今天必须上线"的动力 | 缺乏每日任务/签到 |

## 三、系统详细设计

### 3.1 图鉴收集系统（Collection Book）

**设计目的**：给每一次开果赋予"记录价值"，即便开出空壳也要在图鉴上盖个"已解锁"的章，消除负面情绪。

**图鉴结构**：

| 品种 | 空壳 | 少肉 | 正常 | 满肉 | 爆肉 | 榴莲之王 |
|------|------|------|------|------|------|---------|
| 金枕 | □ | □ | □ | □ | □ | □ |
| 干尧 | □ | □ | □ | □ | □ | □ |
| 猫山王 | □ | □ | □ | □ | □ | □ |

共 3×6 = 18格。每个格子对应一种"品种+出肉率档位"组合。

**解锁规则**：
- 开出对应出肉率 → 自动解锁该格
- 已解锁格显示该次开果的详细信息（日期、售价）
- 未解锁格显示"？"剪影，提示"尚未开出此结果"

**收集奖励**：

| 完成度 | 奖励 |
|--------|------|
| 集齐某品种全部6格 | 该品种"稀有榴莲券"×1（下次购买该品种必出极品外表） |
| 集齐全部18格 | "传说榴莲券"×1 + 专属称号"榴莲博士" |

**UI入口**：主界面底部新增"图鉴"按钮

---

### 3.2 成就系统（Achievements）

**设计目的**：提供中长期的"里程碑式"目标，给玩家阶段性满足感。

**12个成就**：

| ID | 成就名 | 触发条件 | 奖励 |
|----|--------|---------|------|
| A01 | 初出茅庐 | 累计开果10次 | 100金币 |
| A02 | 老练果贩 | 累计开果50次 | 500金币 |
| A03 | 猫山王猎人 | 开出5次猫山王（任意出肉率） | 200金币 |
| A04 | 火眼金睛 | 选购极品外表榴莲且开出爆肉 | 300金币 + 称号"慧眼" |
| A05 | 榴莲之王 | 开出"榴莲之王"评级 | 500金币 + 特效称号"王" |
| A06 | 破产边缘 | 金币低于10 | 救济金200 |
| A07 | 万元户 | 累计金币达到10000 | "金榴莲"头像框 |
| A08 | 连击大师 | 达成5连击 | 200金币 |
| A09 | 图鉴收集者 | 图鉴解锁12格 | 稀有榴莲券×1 |
| A10 | 日常达人 | 完成10次每日任务 | 100金币 |
| A11 | 幸运儿 | 累计触发10次暴击 | 200金币 |
| A12 | 榴莲博士 | 图鉴解锁全部18格 | 传说榴莲券×1 |

**成就弹窗**：
- 达成时全屏弹出，显示成就图标 + 名称 + 奖励
- DOTween 弹入 + 粒子特效
- 3秒后自动消失，或点击关闭

---

### 3.3 连击系统（Combo System）

**设计目的**：把"连续好运"变成可视化、有奖励的爽点，让人想"再来一次"。

**连击规则**：

| 上次出肉率 | 连击判定 |
|-----------|---------|
| 正常 / 满肉 / 爆肉 / 榴莲之王 | 连击 +1 |
| 空壳 / 少肉 | 连击清零 |

**连击奖励**（售卖加成）：

| 连击数 | 售卖加成 | 视觉特效 |
|--------|---------|---------|
| 1 | ×1.0 | 无 |
| 2 | ×1.0 | 小火焰图标 |
| 3 | ×1.2 | 火苗 + 音效"Nice!" |
| 5 | ×1.5 | 烈焰 + 音效"On Fire!" |
| 7 | ×2.0 | 全屏火焰边框 + 音效"Unstoppable!" |
| 10+ | ×3.0 | 彩虹烈焰 + 震动 + "GODLIKE!" |

**显示位置**：开果结算后，在评级图标上方显示连击计数器。

**断连保护**（金币解锁，增加金币出口）：
- 花费 100金币 → 购买"连击保护"（下次开出空壳/少肉时不掉连击）
- 商店可购买，背包中消耗品

---

### 3.4 幸运暴击系统（Critical Luck）

**设计目的**：在任何一次开果中都有"意外惊喜"的可能，让每次滑动都有期待。

**触发规则**：

| 触发条件 | 概率 | 效果 |
|---------|------|------|
| 任意开果 | 8% | "幸运暴击！"出肉率 +10%（直接提升一档） |
| 极品外表榴莲 | 20% | 暴击概率翻倍 |
| 劣质外表榴莲 | 2% | "绝地翻盘！"出肉率直接跳到爆肉 |

**视觉反馈**：
- 开果时，如果触发暴击，壳裂瞬间有金色闪电特效
- "幸运暴击！"大字飘出 + 震屏
- 果肉颜色更加金黄闪耀

**与外观系统的关系**：
- 暴击概率受外观等级影响（极品外表暴击率翻倍）
- 这是"为什么多花2.2倍价格买极品"的额外理由
- 暴击覆盖出肉率数值本身（在概率偏移之后），是二次随机

---

### 3.5 每日任务系统（Daily Quests）

**设计目的**：给玩家每天打开游戏的理由。

**每日任务池**（每天随机3个）：

| 类型 | 示例任务 | 奖励 |
|------|---------|------|
| 购买类 | "购买3颗榴莲" | 50金币 |
| 开果类 | "开出1次大赚以上" | 80金币 |
| 品种类 | "购买1颗猫山王" | 60金币 |
| 售卖类 | "累计售卖获得500金币" | 100金币 |
| 广告类 | "看1次广告" | 30金币 |
| 收集类 | "解锁1个新图鉴格" | 150金币 |
| 连击类 | "达成3连击" | 100金币 |

**刷新规则**：
- 每天00:00刷新3个任务
- 未完成的任务不累积
- 完成全部3个 → 额外奖励：100金币 + 随机Buff
- 看广告可额外刷新1个任务（每日1次）

**UI位置**：主界面顶部新增"每日任务"按钮，显示进度"1/3"

### 4.2 图鉴页

- 3×6 网格，每格显示品种名 + 出肉率档位
- 已解锁：彩色 + 显示最佳记录日期和售价
- 未解锁：灰色剪影 + "？"
- 顶部显示进度条："12/18 已收集"

### 4.3 成就页

- 列表展示12个成就

- 3个任务卡片，每张显示任务描述 + 进度 + 奖励
- 完成的任务显示绿色对勾 + "领取"按钮
- 底部"全部完成奖励"区域

## 五、金币新出口（消耗点）

v1.3 之前金币基本只能用来买榴莲和升商店。新增以下消耗途径：

| 消耗项 | 费用 | 说明 |
|--------|------|------|
| 连击保护道具 | 100金币 | 商店购买，下次断连不掉Combo |
| 幸运符道具 | 200金币 | 下次开果暴击率翻倍（16%） |
| 任务刷新 | 50金币 | 刷新1个每日任务 |
| 商店 Lv.3 升级 | 1,500金币 | 新增：回收价再+15% |

## 六、数据持久化

v1.3 需要持久化的新增数据：

| 数据 | 存储方式 | 说明 |
|------|---------|------|
| 图鉴解锁状态 | PlayerPrefs (18个bool) | 每种组合是否已解锁 |
| 图鉴最佳记录 | PlayerPrefs (18个int) | 每格最佳售价 |
| 成就达成状态 | PlayerPrefs (12个bool) | 每个成就是否已达成 |
| 成就进度 | PlayerPrefs (计数型) | 如"开果次数""猫山王次数" |
| 连击数 | 内存（不持久化） | 当前会话连击 |
| 最高连击 | PlayerPrefs | 历史最高连击 |
| 每日任务状态 | PlayerPrefs + 日期 | 今日任务+进度，跨天重置 |
| 暴击总次数 | PlayerPrefs | 累计暴击次数 |
| 玩家等级 | PlayerPrefs | 经验值累积等级 |
