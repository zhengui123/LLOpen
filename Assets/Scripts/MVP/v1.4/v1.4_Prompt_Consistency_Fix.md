# 《榴莲开了》v1.4 - 榴莲贴图一致性优化文档

> 版本：v1.4 | 日期：2026-06-24
> 适用范围：MVP + v1.2 + v1.3 中所有榴莲相关贴图（约20条提示词）
> 核心改动：建立"主参考 + 增量描述"体系，确保同品种贴图视觉一致

---

## 一、问题根因分析

### 1.1 现象

同一品种（如金枕）的贴图——未开普通、未开劣质、已开空壳、开裂阶段——看起来像**四颗完全不同的榴莲**，形态、角度、颜色、光影各说各话。

### 1.2 根因拆解

| 问题 | 根因 | 案例 |
|------|------|------|
| **无身份锚点** | 每个提示词独立描述一颗"金枕榴莲"，AI 每次从头构思 | D-01 描述"oval shape"，D-A01 描述"slightly squashed shape"——矛盾 |
| **角度不一致** | 有的写"top-down view"，有的不写，AI 随机选择视角 | D-01 是俯视，D-O01 可能变成侧面——拼不起来 |
| **光影不一致** | 有的写"soft natural lighting"，有的写"warm spotlight" | 同品种的不同贴图光源方向/强度不同 |
| **颜色无锁定** | 只用模糊词汇"pale yellowish green"，AI 每次调出不同色 | 三张金枕贴图可能是三种不同的黄绿色 |
| **开裂序列无关联** | CP-01~05 是5个独立提示词，AI 认为是5颗不同的榴莲 | 从 CP-01 到 CP-02 可能连形状都变了 |
| **果肉与壳不匹配** | FL-01（果肉块）和 D-01（完整榴莲）毫无关联 | 果肉颜色和壳内颜色对不上 |

### 1.3 本质

当前的提示词体系是**"描述式"**——每条独立描述对象。需要改为**"主参考 + 增量式"**——先定义一颗榴莲的完整身份，后续所有变体只描述"相对于主参考的增量变化"。

---

## 二、优化方法论 —— "主参考 + 增量"

### 2.1 核心原则

```
主参考（Master Reference）= 锁定不变的部分
增量描述（Delta Description）= 只写变化的部分
```

**主参考锁定的内容**：
1. 品种外形特征（形状、大小比例、刺的特征）
2. 标准颜色（用具体色号锚定，如 #C8D5A3）
3. 观看角度（统一："正上方俯瞰 + 10度前倾"）
4. 光源设定（统一："左上方柔光，色温 4500K"）
5. 构图比例（统一："画面居中，填充80%"）

**增量描述只写变化**：
- 外观等级变化 → 只写壳的色泽/裂纹/叶片变化
- 开裂阶段变化 → 只写裂纹位置/宽度/深度变化
- 已开状态变化 → 只写壳分离程度/果肉暴露量变化

### 2.2 提示词模板

```
【主参考】金枕榴莲 MATER REFERENCE:
[锁定全部固定属性: 形状/颜色/刺/角度/光源/构图]

【变体】金枕 - 劣质外表:
SAME durian as MASTER REFERENCE above, but with the following changes ONLY:
[仅描述变化: 壳色变暗 / 出现裂纹 / 叶片枯萎...]
DO NOT change: shape, angle, lighting, composition, thorn pattern.

【变体】金枕 - 已开空壳:
SAME durian as MASTER REFERENCE above, but now SPLIT OPEN:
[仅描述变化: 壳沿裂纹裂开 / 内部空洞...]
DO NOT change: remaining shell color, angle, lighting, composition.
```

### 2.3 跨提示词关联关键词

在所有变体提示词中强制加入以下锚定语：

```
"This is the EXACT SAME durian as the Master Reference."
"Maintain identical angle, lighting, and composition."
"Only the [specific feature] has changed."
"The underlying shape, thorn pattern, and proportions are UNCHANGED."
```

## 三、三种榴莲的主参考定义

### 3.1 金枕榴莲（Jinzheng）—— MASTER REFERENCE

```
【金枕 MASTER REFERENCE —— 所有金枕贴图以此为基准】

IDENTITY:
- 品种: Jinzheng (金枕), the most common Thai durian variety
- 形状: Slightly oval, 110% height-to-width ratio (taller than wide)
- 壳色: Pale yellowish-green, precise color #C8D5A3, with subtle golden undertone
- 刺: Short, dense, blunt thorns (~1cm length), evenly distributed
- 顶部特征: A single fresh green leaf at the stem, slightly curved rightward
- 底部: Slightly flattened, stable resting shape

COMPOSITION (LOCKED for ALL Jinzheng images):
- 视角: Top-down view with 10-degree forward tilt
- 尺寸: Fills 80% of 512×512 canvas, centered
- 光源: Soft diffused studio light from upper-left (45° angle), warm color temperature 4500K
- 阴影: Subtle soft shadow cast to lower-right, opacity ~30%
- 背景: Pure white (#FFFFFF) or transparent, no ground plane visible

THORN PATTERN (CONSISTENT):
- 顶部区域刺较密, 中部均匀分布, 底部刺稍长
- 刺的弯曲方向: 轻微向外弯曲, 非垂直直立
```

1. **先生成主参考**：先跑 D-01/D-02/D-03 + CP-01 这4条，确认4张图的形态、角度、光影统一后，再跑其余变体
