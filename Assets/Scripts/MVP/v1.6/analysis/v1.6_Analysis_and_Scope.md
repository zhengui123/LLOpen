## 二、v1.6 升级范围定义

### 原则

v1.6 是 **v1.5 的完成态**——把打好草稿但没落地的全部落地，把已知 BUG 修干净。不做新功能。

### v1.6 纳入项（6 个执行单元）

| # | 单元 | 内容 | 预估行数 | 来源 |
|---|------|------|----------|------|
| **U1** | 任务弹层 | `DailyTargetPanel.cs` + 场景节点 `QuestPanel` + 底栏接线 | ~150 行 | P-A1 |
| **U2** | 分享抽象 | `ShareService` + `IShareProvider` + `EditorShareProvider` + `DurianOpener` 改造 | ~200 行 | P-A2 |
| **U3** | 场景空引用修复 | ShareButtonGroup 节点、refreshButtonText、coinParticleRoot、upgradeEffectRoot | 编辑器操作 | P-F1~F5 |
| **U4** | 贴图占位绑定 | CB/CO 四个字段用已有贴图占位 → 图鉴不再纯色块 | 编辑器操作 | P-A3 |
| **U5** | v1.2 废弃代码清理 | 删 `UpdateCrackProgress` 残留、半壳逻辑、D-O 引用、CP 引用 | ~50 行 | 技术债 |
| **U6** | 编译 & Play 验证 | 全量编译无 Error、主流程 Play 无 Exception | — | 质量门 |

### v1.6 明确不纳入

| 不纳入 | 原因 |
|--------|------|
| 微信 SDK 真实分享（P-B） | 依赖微信小游戏工程/资质，当前不具备 |
| 分享回流奖励（P-C1） | 需先有分享 SDK |
| 顶栏动效统一（P-C2） | 低优先级视觉，放 v1.7 |
| KnifeTool 音效（P-F2） | 项目内无标准裂壳音频资源 |
| 新榴莲品种 RC 壳盖 | 干尧/猫山王壳盖需新 AI 出图，放 v1.7 |
| 成就系统复活 | 已砍，不复活 |

## 五、v1.6 验收清单

### 功能验收（Play 模式）

- [ ] 底栏「任务」点开 → 弹层显示每日目标进度 500 金，与顶栏数字一致
- [ ] 达标未领 → 弹层与顶栏均可领 +50 金，领后两处同步
- [ ] 开果/售卖页不显示底栏，任务入口不可达
- [ ] 爆肉或连击≥3 → 出现「炫耀一下」按钮
- [ ] 点击「炫耀一下」→ Console 打印结构化分享文案 + 按钮反馈
- [ ] Editor 下不走微信 SDK，不报错
- [ ] 图鉴页背景不再纯色块（CB-01 占位后）
- [ ] 连击火焰有 CO-01 占位图
- [ ] Console 无新增 Error / Exception

### 代码质量

- [ ] `Assets/_Project/Scripts/` 全量编译通过，0 Error
- [ ] 无 `using` 未引用、无 dead code 警告
- [ ] 所有 `[Inject]` 字段在 `GameLifetimeScope.Configure` 中有对应注册
