## U1：任务弹层 DailyTargetPanel

### 目标

底栏「任务」按钮不再 `Debug.Log`，改为打开轻量弹层：显示每日目标「今日卖出累计 500 金」的规则、进度条、领奖状态。

### 技术方案（来自 `v1.5_打磨计划.md` 4.2-4.3）

- **新脚本**：`DailyTargetPanel.cs`，位于 `Assets/_Project/Scripts/View/`
- **场景节点**：`MainShell/QuestPanel`（默认 `SetActive(false)`）
- **数据源**：只读 `DailyTarget`，领奖调用 `ClaimReward()`
- **导航**：底栏 Quest → 打开弹层；关闭后停留当前 Hub 页
- **UI 布局**：

```
┌─────────────────────────────┐
│  [×]  今日目标               │
│  ─────────────────────────   │
│  今日卖出累计达到 500 金币    │
│  [========····] 320/500      │
│  奖励：50 金币               │
│  [ 领取 +50 ]  （达标后可点） │
│  明日 0 点重置               │
└─────────────────────────────┘
```

### Cursor 提示词

```
创建 Assets/_Project/Scripts/View/DailyTargetPanel.cs：

===== 类设计 =====
using UnityEngine;
using UnityEngine.UI;
using VContainer;

public class DailyTargetPanel : MonoBehaviour
{
    [Inject] private DailyTarget dailyTarget;
    [Inject] private GameUIRoot gameUIRoot; // 可选，获取金币图标

    [SerializeField] private GameObject panelRoot;
    [SerializeField] private Button closeButton;
    [SerializeField] private Text progressText;       // "320/500"
    [SerializeField] private Slider progressBar;       // fill = 320/500
    [SerializeField] private Text rewardText;           // "奖励：50 金币"
    [SerializeField] private Button claimButton;
    [SerializeField] private Text claimButtonText;      // "领取 +50" / "已领取"
    [SerializeField] private Text resetHintText;        // "明日 0 点重置"

    private void Start()
    {
        closeButton.onClick.AddListener(Hide);
        claimButton.onClick.AddListener(OnClaimClicked);
        panelRoot.SetActive(false);
    }

    public void Show()
    {
        Refresh();
        panelRoot.SetActive(true);
    }

    public void Hide()
    {
        panelRoot.SetActive(false);
    }

    private void Refresh()
    {
        int earned = dailyTarget.EarnedToday;
        int target = dailyTarget.TargetGold;
        bool completed = dailyTarget.IsCompleted;
        bool claimed = dailyTarget.IsClaimed;

        progressText.text = $"{earned}/{target}";
        progressBar.maxValue = target;
        progressBar.value = Mathf.Min(earned, target);
        rewardText.text = $"奖励：{dailyTarget.REWARD} 金币";

        if (claimed)
        {
            claimButton.interactable = false;
            claimButtonText.text = "已领取";
        }
        else if (completed)
        {
            claimButton.interactable = true;
            claimButtonText.text = $"领取 +{dailyTarget.REWARD}";
        }
        else
        {
            claimButton.interactable = false;
            claimButtonText.text = $"还需 {target - earned} 金";
        }
    }

    private void OnClaimClicked()
    {
        if (dailyTarget.IsCompleted && !dailyTarget.IsClaimed)
        {
            dailyTarget.ClaimReward();
            Refresh();
            // 通知顶栏刷新
            // EventBus 或直接调 gameUIRoot 更新金币显示
        }
    }
}

===== 注册到 GameLifetimeScope =====
修改 Assets/_Project/Scripts/Core/GameLifetimeScope.cs：
在 Configure 中添加 builder.RegisterComponentInNewPrefab(...) 或手动绑定场景中的 DailyTargetPanel。

推荐：在 Main.unity 的 MainShell 下挂 DailyTargetPanel，用 builder.RegisterComponentInHierarchy<DailyTargetPanel>() 注册。

===== 修改 MainShellUI.cs =====
在底栏「任务」按钮的 onClick 中：

其中 dailyTargetPanel 通过 [Inject] 或 [SerializeField] 引用。

===== 场景节点搭建 =====
在 MainShell 下创建 QuestPanel（GameObject，挂 DailyTargetPanel.cs）：
- panelRoot → QuestPanel 自身
- closeButton → 右上角 × 按钮
- progressText → TMP_Text 或 Text
- progressBar → Slider（背景灰色，填充金色）
- rewardText → "奖励：50 金币"
- claimButton → Button，子 Text "领取 +50"
- claimButtonText → 按钮子 Text
- resetHintText → "明日 0 点重置"
- 默认 SetActive(false)

底栏「任务」按钮绑定 → 打开 QuestPanel

===== 底栏徽章更新 =====
MainShellUI 刷新顶栏/底栏时：
- 读取 dailyTarget.EarnedToday / TargetGold
- 未完成 → 显示 "n/500"
- 达标未领 → 显示 "!"
```

## U2：分享抽象 ShareService

### 目标

`DurianOpener` 不再直接 `Debug.Log`，改为调用 `ShareService.TryShare(ShareContext)`。Editor 下走 `EditorShareProvider`（Console 打印 + 按钮反馈），微信环境走真 SDK（Phase B 后续接）。

### 架构（来自 `v1.5_打磨计划.md` 5.2）

```
DurianOpener.OnShareClicked()
    → ShareService.TryShare(ShareContext)
        → IShareProvider
            → EditorShareProvider      （Editor / Standalone）
            → WeChatMiniGameShareProvider（微信 WASM，Phase B）
```

### Cursor 提示词

```
===== 文件 1：Assets/_Project/Scripts/Model/ShareContext.cs =====
namespace LLOpen
{
    public struct ShareContext
    {
        public string VarietyName;       // "金枕"
        public string YieldGradeName;    // "爆肉"
        public int EstimatePrice;        // 估价
        public int ComboCount;           // 连击数
        // 可选：Texture2D screenshot;

        public string ToShareText()
        {
            var comboStr = ComboCount >= 3 ? $"，{ComboCount}连击" : "";
            return $"我在榴莲大亨开出了{YieldGradeName}{VarietyName}{comboStr}，估价 {EstimatePrice} 金！";
        }
    }
}

===== 文件 2：Assets/_Project/Scripts/Platform/IShareProvider.cs =====
namespace LLOpen
{
    public interface IShareProvider
    {
        /// <summary>返回 true 表示分享成功</summary>
        bool TryShare(ShareContext ctx);
    }
}

===== 文件 3：Assets/_Project/Scripts/Platform/EditorShareProvider.cs =====
using UnityEngine;

namespace LLOpen
{
    public class EditorShareProvider : IShareProvider
    {
        public bool TryShare(ShareContext ctx)
        {
            Debug.Log($"[Share] {ctx.ToShareText()}");
            return true; // Editor 下模拟成功
        }
    }
}

===== 文件 4：Assets/_Project/Scripts/System/ShareService.cs =====
using VContainer;

namespace LLOpen
{
    public class ShareService
    {
        private readonly IShareProvider provider;

        public ShareService(IShareProvider provider)
        {
            this.provider = provider;
        }

        public bool TryShare(ShareContext ctx)
        {
            return provider.TryShare(ctx);
        }
    }
}

===== 注册到 GameLifetimeScope =====
修改 Assets/_Project/Scripts/Core/GameLifetimeScope.cs：

// 根据编译宏选择 Provider
#if UNITY_EDITOR || UNITY_STANDALONE
    builder.Register<IShareProvider, EditorShareProvider>(Lifetime.Singleton);
#elif UNITY_WEBGL && WEIXINMINIGAME
    // builder.Register<IShareProvider, WeChatMiniGameShareProvider>(Lifetime.Singleton);
    // 当前 Phase B 未开发，fallback 到 Editor
    builder.Register<IShareProvider, EditorShareProvider>(Lifetime.Singleton);
#else
    builder.Register<IShareProvider, EditorShareProvider>(Lifetime.Singleton);
#endif

builder.Register<ShareService>(Lifetime.Singleton);

===== 修改 DurianOpener.cs =====
1. 注入 ShareService：
   [Inject] private ShareService shareService;

2. 找到 OnShareClicked / 炫耀一下按钮的 onClick：
   - 旧：Debug.Log("分享");
   - 新：
     var ctx = new ShareContext
     {
         VarietyName = currentDurian.variety.ToString(),
         YieldGradeName = GetYieldGradeName(finalGrade),  // 用已有的评级转换
         EstimatePrice = currentEstimate,
         ComboCount = streakCounter.CurrentStreak
     };
     bool success = shareService.TryShare(ctx);
     
     // UI 反馈
     shareButtonText.text = success ? "已分享" : "分享失败";
     // DOTween 0.3s 淡出后恢复
     shareButtonText.DOFade(0f, 0.3f).SetDelay(1.5f).OnComplete(() => 
     {
         shareButtonText.text = "炫耀一下";
         shareButtonText.DOFade(1f, 0f);
     });

3. 可选：发送 EventBus.ShareSucceededEvent：
   if (success) EventBus.Publish(new ShareSucceededEvent { Context = ctx });
```

3. 按钮短暂变为「已分享」→ 1.5s 后恢复「炫耀一下」

## U4：贴图占位绑定

### 目标

CB-01~03、CO-01 四个 `DurianSpriteConfig` 空字段用已有贴图占位，消除图鉴/连击的纯色块表现。

### 操作（编辑器 Inspector，不需 Cursor 代码）

| 字段 | 占位用贴图 | 路径 |
|------|-----------|------|
| `collectionBookBg` (CB-01) | `marketFrame` (F-01) | 拖入同一 Sprite 引用 |

| `collectionCompleteStamp` (CB-03) | `perfectRating` (R-05) | 拖入同一 Sprite 引用 |
